ten reserved JS keyword and their action

  1. break:
    The break statement is used to terminate a loop, switch, or label statement. When encountered, the control of the program moves out of the loop.
  2. case:
    The case keyword is used in switch statements to define various possible values that switch can match. If the value matches, the associated block of code is executed.
  3. catch:
    The catch block is used in conjunction with try to handle exceptions. If an error occurs in the try block, the catch block executes.
  4. class:
    The class keyword is used to define a class. Classes are templates for creating objects and can include methods and constructors.
  5. continue:
    The default keyword is used in switch statements to specify the code that should run if no case matches the switch expression.
  6. delete:
    The delete keyword is used to remove a property from an object.
  7. finally:
    The finally block is used in conjunction with try and catch to execute code regardless of whether an error occurred or not.
  8. function:
    The function keyword is used to declare a function.
  9. return:
    The return statement is used to exit a function and return a value.

These keywords are essential to understanding and writing JavaScript code effectively.
